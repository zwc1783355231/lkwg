# 洛克王国世界本地地图工具

这是一个面向 Windows 的本地地图工具，用来查看和管理《洛克王国世界》的地图点位数据，并提供两套界面实现：

- `desktop_map_tool.py`：Tkinter 版本 （不建议，这玩意太臃肿了，codex阴我）
- `desktop_map_tool_qt.py`：Qt 版本

项目内已经包含地图图片、点位数据和图标资源，启动后可以直接在本地使用，不依赖在线服务。

## 功能概览

- 浏览地图点位与分类数据
- 显示采集物、地点等大类信息
- 支持地图缩放、透明度、描边颜色、路径颜色等界面配置
- 支持小地图区域定位与跟踪相关设置
- 提供调试参数和状态保存

## 运行环境

建议在 Windows 上使用。脚本依赖如下：（其实可以直接用exe，虽然我不确定exe有没有保存配置的功能）

- Python 3.10 或更高版本
- PyQt5（Qt 版本需要）
- Pillow
- OpenCV Python
- NumPy

Tkinter 属于 Python 标准库，Tkinter 版本不需要额外安装。

## 启动方式

### Tkinter 版本

双击 `run_desktop_map.bat`，或在项目根目录执行：

    python desktop_map_tool.py

### Qt 版本

双击 `run_desktop_map_qt.bat`，或在项目根目录执行：

    python desktop_map_tool_qt.py

如果你使用的是 Anaconda 环境，批处理文件会自动调用基础环境。若你自己的 Python 已经配置好，也可以直接运行脚本。

## 配置文件

项目会读取并保存以下配置文件：

- `app_config.json`：Tkinter 版本的窗口和跟踪参数
- `qt_app_config.json`：Qt 版本的界面和布局参数

如果配置被改乱，可以直接删除对应文件，程序会按默认值重新生成。

## 数据目录

主要数据都放在 `data/` 目录下：

- `map_8192.png`：主地图图片
- `map_match_*.png`：不同尺寸的匹配地图缓存
- `01_地点.json`：地点数据
- `02_可刷新采集物.json`：可刷新采集物数据
- `03_一次性采集物.json`：一次性采集物数据
- `image/`：点位图标资源

补充资源放在 `resources/` 目录下，临时数据和运行时状态放在 `temp/`、`user_data/`。

## 打包

项目提供了 PyInstaller 配置文件：

- `lkwg_map_qt.spec`
- `lkwg_map_qt_onefile.spec`

这两个配置都以 `desktop_map_tool_qt.py` 作为入口，并把 `data/`、`resources/`、`app_config.json`、`qt_app_config.json` 打包进去。

## 目录结构

    lkwg/
    ├─ desktop_map_tool.py
    ├─ desktop_map_tool_qt.py
    ├─ run_desktop_map.bat
    ├─ run_desktop_map_qt.bat
    ├─ app_config.json
    ├─ qt_app_config.json
    ├─ data/
    ├─ resources/
    ├─ temp/
    └─ user_data/

## 说明

- 这是一个本地工具，所有数据都保存在当前目录。
- 运行过程中生成的缓存文件通常放在 `temp/`。
- 界面状态、选择记录等用户数据放在 `user_data/`。
